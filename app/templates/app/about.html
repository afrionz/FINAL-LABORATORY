About the System

The Campus Facility Management System is a web-based application designed to help students, faculty, and staff efficiently manage campus facilities. The system allows users to view available facilities, submit reservation requests, and report maintenance concerns in an organized and user-friendly platform.

Objectives

  • To provide an organized system for reserving campus facilities
  • To reduce manual paperwork and scheduling conflicts
  • To allow users to submit maintenance requests online
  • To improve efficiency in facility management

Key Features

  • Facility reservation with date and time scheduling
  • Real-time facility availability viewing
  • Maintenance request submission
  • Administrative management via Django Admin
  • User-friendly and responsive interface

Target Users

  • Students
  • Faculty Members
  • Administrative Staff
  • Campus Facility Managers

Technologies Used

  • Django (Python Web Framework)
  • HTML, CSS, and JavaScript
  • SQLite Database
  • Bootstrap / Custom CSS
